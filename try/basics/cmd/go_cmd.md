# go 命令行

| 命令| 描述 |
|----------|-----------------------------------------------------|
| bug      | 启动bug报告                                           |
| build    | 编译包及依赖                                           |
| clean    | 删除中间文件                                          |
| doc      | 显示包或符号的文档                                     |
| env      | 打印Go环境信息                                         |
| fix      | 更新包以使用新的API                     |
| fmt      | 在包资源上执行 gofmt                  |
| generate | 通过处理源文件生成Go文件           |
| get      | 下载并安装包及依赖 |
| install  | 编译并安装包及依赖   |
| list     | 列出包                   |
| mod      | 扩展模块版本                                 |
| run      | 编译并执行Go程序                 |
| test     | 测试包                           |
| tool     | 执行指定的go工具                        |
| version  | 打印Go版本                          |
| vet      | 在包上执行go tool vet              |
